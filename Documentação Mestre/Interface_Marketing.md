# Interface do Usuário: Marketing

**Nível de Acesso:** Intermediário
**Responsável:** Comunicação, redes sociais e identidade visual dos produtos.

## Ferramentas Disponíveis no Telegram (Botões Principais)

1. **📢 Campanhas:**
   - **Descrição:** Ferramentas para gerar textos de divulgação.
   - **Comportamento:** Pode usar a IA do bot para gerar legendas focadas em vendas para o Instagram/WhatsApp com base nos produtos em estoque.

2. **Gerar Link de Acesso (Catálogo Web):**
   - **Descrição:** Acesso restrito à "Página de Ajustes".
   - **Comportamento:** Gera o Link Mágico (JWT) via Telegram.

## Permissões na "Página de Ajustes de Catálogo" (Backend Web)
- **Visualizar:** Todos os produtos que estão no status `PENDENTE` ou `PUBLICADO`.
- **Editar Foco Visual:** A prioridade do Marketing é ajustar Descrições (textos apelativos/SEO) e a qualidade das Imagens.
- **Ação Restrita:** Não pode alterar preços sem autorização, nem publicar diretamente no site. O trabalho é focado no polimento visual.
