# Interface do Usuário: Boss

**Nível de Acesso:** Alto (Gestão)
**Responsável:** Supervisão da loja, acompanhamento de metas e relatórios.

## Ferramentas Disponíveis no Telegram (Botões Principais)

1. **📊 Relatórios:**
   - **Descrição:** Acesso às métricas de desempenho.
   - **Comportamento:** O Boss pode solicitar o resumo de produtos cadastrados, produtos mais clicados e métricas de engajamento do quiosque.

2. **Gerar Link de Acesso (Catálogo Web):**
   - **Descrição:** Acesso à "Página de Ajustes".
   - **Comportamento:** Gera o Link Mágico (JWT) via Telegram.

## Permissões na "Página de Ajustes de Catálogo" (Backend Web)
- **Visualizar:** Acesso completo a toda a lista de produtos.
- **Editar:** Permissão para corrigir textos, descrições e validar o trabalho feito pelos níveis operacionais (Quiosque/Marketing).
- **Ação Restrita:** O Boss **não** publica no site automaticamente. Ele prepara e audita o lote de produtos e repassa ao Admin para o clique final de `Publicação`.
